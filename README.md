SmartPark: Sistema de parqueo inteligente en C# (Consola) para gestión de tickets y cobros. Calcula tarifas por tiempo y tipo de vehículo, aplica multas, recargos y descuentos VIP. Simula el tiempo y valida entradas, usando solo variables simples y un ticket activo según restricciones académicas.
